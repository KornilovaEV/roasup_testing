# Tестовое задание на позицию JavaScript Разработчик (Playable Ads)

> Общее описание 
* Необходимо разработать Playable AD. 
* Цель рекламы - мотивировать пользователя установить игру.

## Для запуска проекта локально надо:
- npm install
- npm start

### Выполнено из доп. 

* Использование Pixi.js  v."8.14.3"
* Использование Webpack v.5
* Использование Babel
